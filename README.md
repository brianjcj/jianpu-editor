# 简谱编辑器 (JianPu Editor)

一个基于 Web 的简谱编辑工具，支持文本输入、实时渲染、音乐播放和打印导出。

## 功能特点

- 📝 **简谱标记语法** - 快速输入简谱，类似 Markdown 的简洁语法
- 🎼 **实时渲染** - Canvas 高清渲染，所见即所得
- 🖨️ **A4 打印** - 适配 A4 纸尺寸，可直接打印
- 💾 **PNG 导出** - 导出高清乐谱图片
- ▶️ **音乐播放** - Web Audio API 合成播放
- 📱 **响应式设计** - 支持桌面和移动端
- 🎛️ **面板切换** - 可自由控制编辑器/语法提示/预览区的显示

## 快速开始

直接在浏览器中打开 `index.html` 即可使用，无需安装。

## 简谱标记语法

| 标记 | 说明 |
|------|------|
| `1-7` | 中音 do re mi fa sol la si |
| `^1-^7` | 高音（上方加点） |
| `_1-_7` | 低音（下方加点） |
| `-` | 延长音/休止符 |
| `\|` | 小节线 |
| `\|\|` | 终止线 |
| `1/` | 八分音符（下划线） |
| `1//` | 十六分音符（双下划线） |
| `1.` | 附点音符 |
| `1#` | 升半音 |
| `1b` | 降半音 |
| `!标题` | 设置标题 |
| `%C=4/4` | 设置调号和拍号 |
| `@100` | 设置速度（BPM） |

## 示例

```
!小星星
%C=4/4
@100

1 1 5 5 | 6 6 5 - | 4 4 3 3 | 2 2 1 - |
5 5 4 4 | 3 3 2 - | 5 5 4 4 | 3 3 2 - |
1 1 5 5 | 6 6 5 - | 4 4 3 3 | 2 2 1 - ||
```

## 技术栈

- HTML5 Canvas 2D
- Web Audio API
- 纯前端实现，无需后端
